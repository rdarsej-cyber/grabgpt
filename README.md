# GrabGPT

Export your ChatGPT conversations. One command. Fully local.

## Install

**Windows (PowerShell):**
```powershell
irm grabgpt.darsej.com/install.ps1 | iex
```

**Linux/macOS:**
```bash
curl -fsSL grabgpt.darsej.com/install.sh | sh
```

**Or if you already have Node.js:**
```bash
npx grabgpt@latest
```

## How It Works

1. Run the command — a browser tab opens automatically
2. Paste your ChatGPT auth token (guide included in the app)
3. Select which conversations to export
4. Download as Markdown, JSON, or both

## Privacy

- Everything runs on your machine
- Your data never leaves localhost
- No analytics, no telemetry, no external calls
- Fully open source — read every line
