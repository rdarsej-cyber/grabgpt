How to get your auth token
  1. Open chatgpt.com in your browser and make sure you are logged in.
  2. Open the browser developer tools: press F12 (Windows/Linux) or Cmd+Option+I (Mac), then go to the Network tab.
  3. Refresh the page, then click any network request to chatgpt.com.
  4. In the Request Headers section find Authorization. Copy the value starting with Bearer ey... — paste only the token part (after "Bearer ").
  5. Optionally copy the full Cookie header value and paste it into the Cookies field below to improve reliability.
